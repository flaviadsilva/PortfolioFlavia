# 🚀 Portfólio Profissional | Flávia Silva

> **Status do Projeto:** 🚀 Concluído / Em constante evolução

Este é o repositório do meu portfólio profissional, desenvolvido para apresentar meus projetos como Desenvolvedora Front-End, minhas competências técnicas e meu diferencial em liderança e gestão de pessoas. O foco principal da página é a conversão de clientes PJ e grandes empresas que buscam soluções web robustas e de alto impacto.

🔗 **Acesse o site ao vivo:** [flaviadsilva.github.io/PortfolioFlavia/](https://flaviadsilva.github.io/PortfolioFlavia/)

---

## 🎯 Objetivo do Projeto

Criar uma Landing Page de página única (Single Page) com um design moderno, técnico e corporativo. O objetivo é demonstrar como consigo unir a precisão do desenvolvimento de interfaces com a habilidade de liderar processos e gerenciar expectativas, gerando valor real para o negócio dos clientes.

---

## 🧠 Seções do Site

* **Home / Headline:** Apresentação de impacto focada no valor entregue ao cliente.
* **Sobre Mim:** Minha trajetória na área de TI, paixão por desenvolvimento web e habilidades interpessoais.
* **Projetos em Destaque:** Demonstração prática de problemas resolvidos, metodologias aplicadas e resultados alcançados.
* **Serviços e Habilidades:** Divisão clara entre *Hard Skills* (tecnologias) e *Soft Skills* (liderança e gestão).
* **Contato (CTA):** Canais de comunicação direta para propostas e parcerias.

---

## 🛠️ Tecnologias e Ferramentas Utilizadas

No desenvolvimento deste projeto, utilizei as seguintes tecnologias para garantir um código limpo, responsivo e performático:

* **HTML5** - Estruturação semântica do conteúdo.
* **CSS3** - Estilização moderna e layout responsivo.
* **JavaScript** - Interatividade e dinamismo na página.
* **Git & GitHub** - Controle de versão e hospedagem através do GitHub Pages.
